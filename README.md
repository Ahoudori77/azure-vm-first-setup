# 初めてのAzure VM作成記録

## 概要
Azureネットワーク学習の一環として、`rg-azure-web`リソースグループ内に仮想ネットワークとVMを作成した記録です。

## 実施内容
- RGの作成
- VNetとSubnetの作成
- Public IPアドレスの作成
- NSGの設定
- 仮想マシン（Ubuntu）作成

## 使用ツール
- Azureポータル
- draw.io（構成図作成）

## 補足
このプロジェクトは、Qiita記事「初めてのAzure VM構築記録」と連携しています。
